# My First Spring Boot REST API

Простой REST API на Spring Boot с базовыми endpoint'ами.

## Endpoints:
- `GET /hello` - Возвращает приветствие
- `GET /time` - Текущее время
- `GET /greet/{name}` - Приветствие по имени

## Технологии:
- Java 17
- Spring Boot 3.4.12
- Spring Web

## Как запустить:
1. Клонируй репозиторий
2. Запусти `DemoApplication.java`
3. Открой http://localhost:8080/hello
